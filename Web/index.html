Syllabus

This course examines advanced rendering topics in computer graphics. It extends course 605.667 Computer Graphics which focused on the principles of computer graphics. This course focuses on larger, more complex graphics applications. Topics include: surface representations including parametric representations, subdivision surfaces, and procedural methods. Advanced texture mapping techniques and shadow generation are examined and programmable shaders are extended. Advanced techniques such as ray-tracing and radiosity are discussed. Practical application of these concepts is emphasized through laboratory exercises, culminating in a final project. The course includes laboratory exercises using the C++ programming language and OpenGL on a PC.

Prerequisites

This course is designed to extend 605.667 Computer Graphics. 605.667 is a prerequisite for this course. Students are expected to be familiar with linear algebra (vectors and matrices), modeling and viewing transformations, projections, lighting and shading, image texture mapping, and other “principle” graphics techniques. Also, familiarity with OpenGL is expected.

Laboratory

This course is heavily weighted towards hands-on implementation of the concepts developed in class. There will be a good deal of independent project work. Class lecture time will include a discussion of lab exercises and graphics programming. Lab exercises will use a PC, Microsoft Visual C++, and OpenGL. You should be able to use other compilers as long as you can create standalone applications (e.g. static linking) so that they can be run without using specific DLLs. Graphics exercises use OpenGL 3 with GLSL, GL3W, and FreeGLUT.  We will discuss efficiency and will investigate processing requirements for various rendering tasks.
